To install TestBox just drop it in your web root as '/testbox' orcreate a mapping in your CFML administrator or Application.cfc that points to thedirectory you installed TestBox and create the mapping '/testbox' that points to it.Please note that some TestBox runners require HTML/JavaScript/CSS assets and must beweb accessible.  Therefore, the '/testbox' mapping is necessary for some UI elements.********************************************************************************SYSTEM REQUIREMENTS********************************************************************************- Railo 4.1 (xUnit + BDD)- ColdFusion 10+ (xUnit + BDD)- ColdFusion 9.01+ (xUnit style Only)----AS ALWAYS, VISIT THE WIKI FOR THE LATEST DOCUMENTATION == THE DAILY BREAD == "I am the way, and the truth, and the life; no one comes to the Father, but by me (JESUS)" John 14:1-12
